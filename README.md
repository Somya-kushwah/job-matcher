# 🎯 Job Matcher & Resume Analyzer

A full-stack web application that allows users to upload their resumes and match them with job descriptions using NLP-based keyword ranking.

---

## 🚀 Features

* 🔐 User Authentication (Login / Signup)
* 📄 Resume Upload & Parsing (Apache Tika)
* 🧠 Resume ↔ Job Description Matching
* 📊 Match Score Calculation
* ❌ Missing Skills Detection
* 💼 Job Listings Interface
* ✨ Modern UI (React)

---

## 🧠 How It Works

1. User uploads resume
2. User selects a job OR pastes custom job description
3. Backend extracts resume text using Apache Tika
4. NLP algorithm compares resume with job description
5. System returns match score and missing skills
